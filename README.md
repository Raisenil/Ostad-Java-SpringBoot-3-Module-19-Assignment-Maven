# Ostad Java SpringBoot 3 Module 19 Assignment

### Git Commands

- `git init` - Initialize git repository
- `git add .` - Add all files to staging
- `git commit -m "Init"` - Commit initial code
- `git remote add origin <GitHub URL>` - Add remote repository
- `git push -u origin main` - Push to GitHub
- `git add .` - Add changes after creating Person class
- `git commit -m "Add Person class"` - Commit Person class
- `git push` - Push changes
- `git add README.md` - Add README
- `git commit -m "Add README with commands"` - Commit README
- `git push` - Push README
